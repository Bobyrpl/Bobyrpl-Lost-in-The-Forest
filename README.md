# Lost-in-The-Forest
Game aksi survival 2D berbasis browser. Kamu tersesat di hutan luas dan harus menemukan portal jalan keluar sambil membasmi makhluk-makhluk aneh yang berkeliaran di sekitarmu.

Cara Main


Gerak: W A S D atau tombol panah (desktop) / joystick analog di kiri layar (mobile)
Serang: SPASI (desktop) / tombol ⚔️ di kanan layar (mobile)
Tujuan: cari portal hijau yang berpendar di suatu tempat di peta untuk kabur dari hutan
Peta kecil (minimap): ada di pojok kanan bawah, menunjukkan posisimu, pohon, makhluk, dan lokasi portal keluar# Bobyrpl-Lost-in-The-Forest
